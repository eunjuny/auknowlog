## Q1. 쿠버네티스는 무엇인가요?
- A. 애플리케이션 개발 언어
- B. 리눅스 운영체제
- C. 컨테이너화된 워크로드와 서비스를 관리하기 위한 오픈소스 시스템
- D. 데이터베이스 관리 시스템
✅ 내 답: 컨테이너화된 워크로드와 서비스를 관리하기 위한 오픈소스 시스템
설명: 쿠버네티스는 컨테이너 오케스트레이션 도구로, 컨테이너화된 애플리케이션의 배포, 스케일링 및 관리를 자동화합니다.

## Q2. 쿠버네티스 'kubelet'의 역할은 무엇인가요?
- A. 클러스터의 API를 노출
- B. 각 노드에서 실행되며, 파드 컨테이너가 노드에서 실행되도록 보장
- C. 클러스터 상태를 데이터베이스에 저장
- D. 컨트롤 플레인의 모든 컴포넌트를 관리
✅ 내 답: 각 노드에서 실행되며, 파드 컨테이너가 노드에서 실행되도록 보장
설명: Kubelet은 각 워커 노드에서 실행되는 에이전트로, API 서버로부터 파드 명세를 전달받아 해당 파드의 컨테이너들이 노드에서 정상적으로 실행되고 건강하게 유지되도록 관리합니다.
