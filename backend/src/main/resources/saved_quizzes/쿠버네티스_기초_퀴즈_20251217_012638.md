# 쿠버네티스 기초 퀴즈
총 4문항 · 정답 4 · 오답 0

## Q1. 쿠버네티스의 주된 역할은 무엇인가요?
- A. 가상 머신 관리
- B. 컨테이너 오케스트레이션
- C. 데이터베이스 관리 시스템
- D. 운영체제 개발
✅ 내 답: 컨테이너 오케스트레이션
설명: 쿠버네티스는 컨테이너화된 워크로드와 서비스를 관리하고 자동화하기 위한 오픈소스 시스템입니다. 주로 컨테이너 오케스트레이션 역할을 수행합니다.

## Q2. 쿠버네티스에서 'Pod'는 무엇을 의미하나요?
- A. 물리 서버의 한 종류
- B. 가장 작은 배포 단위
- C. 네트워크 인터페이스 카드
- D. 클러스터 스토리지 볼륨
✅ 내 답: 가장 작은 배포 단위
설명: Pod는 쿠버네티스에서 배포할 수 있는 가장 작은 컴퓨팅 단위로, 하나 이상의 컨테이너를 포함할 수 있습니다.

## Q3. 쿠버네티스 클러스터와 상호작용하기 위한 공식 명령줄 도구는 무엇인가요?
- A. docker-compose
- B. npm
- C. kubectl
- D. git
✅ 내 답: kubectl
설명: kubectl은 사용자가 쿠버네티스 API 서버에 명령을 전송하여 클러스터를 관리할 수 있게 해주는 명령줄 도구입니다.

## Q4. 쿠버네티스에서 'Deployment' 오브젝트의 주된 역할은 무엇인가요?
- A. 영구 스토리지 관리
- B. 클러스터 네트워크 정책 정의
- C. 스테이트리스 애플리케이션의 배포 및 업데이트 관리
- D. 물리 서버의 리소스 할당
✅ 내 답: 스테이트리스 애플리케이션의 배포 및 업데이트 관리
설명: Deployment는 ReplicaSet을 관리하며, Pod의 선언적 업데이트를 담당하여 스테이트리스 애플리케이션을 안정적으로 배포하고 업데이트할 수 있도록 합니다.
